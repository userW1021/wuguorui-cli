# wgr-cli
![Npm 版本](https://img.shields.io/badge/wgr-cli_v0.0.1%20-green)
## 简介

-用于快速搭建前端项目的命令行工具

##模板
- 快速生成Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模板

##  安装

```bash
npm install wgr-cli-tool -g
```
## 使用

```bash

#使用模板创建新项目

wgr create

# 执行项目文件夹名称并选择模板创建新项目

wgr create [projectName]

# 查看脚手架版本
wgr -v
wgr --version

```
